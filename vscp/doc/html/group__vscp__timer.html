Detailed Description

The timer interface is used only by the core, except the processing function vscp_timer_process().

Call the processing function to handle all created timers. If the timers are handled in an interrupt service routine or a different task, than the one which calls vscp_core_process(), don't forget to make the timer functions reentrant.
