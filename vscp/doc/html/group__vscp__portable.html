Detailed Description

Here is all application depended stuff, which can be handled different in any system.

Supported compile switches:

  • VSCP_CONFIG_BOOT_LOADER_SUPPORTED
  • VSCP_CONFIG_IDLE_CALLOUT
  • VSCP_CONFIG_ERROR_CALLOUT

void vscp_portable_bootLoaderRequest ( void  )

This function requests a jump to the bootloader.

It requests it and doesn't expect that it will be immediately. Because the application needs time to change to a safe state before.
