#define VSCP_CLASS_L1_L2_BASE   512

The base id for all level 1 events, sent over level 2.

An event with a class >= 512 but < 1024 will be sent to all Level II clients and to the correct i/f (the one that have the addressed GUID). A response from the device will go out as as a Level II event using the GUID of the interface but class will always have a value < 512 for a response event just as all events originating from a device.
